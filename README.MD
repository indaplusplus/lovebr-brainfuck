## Usage

### Interpreter
```
bfr hello_world.bf <memory_size>
```
`<memory_size>` is the size of the memory in bytes (default: 1024).

### Compiler
```
bfc hello_world.bf hello_world.c <memory_size>
```
`<memory_size>` is the size of the memory in bytes (default: 1024).
